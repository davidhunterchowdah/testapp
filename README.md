# Python Connectivity Tester

A **lightweight web application** for testing server connectivity and serving as a placeholder for production applications. Built with **Python + Flask**, fully containerized with **Docker** and **Docker Compose**.

---

## Features

- Health check endpoint: `/api/health`
- Placeholder API endpoint: `/api/placeholder`
- Optional simple frontend: `/` serves an HTML page
- Lightweight and easy to deploy
- Docker-ready for quick containerization

---

## Requirements

- [Python 3.12+](https://www.python.org/)
- [Docker](https://www.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/)
